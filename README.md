# [Laravel](https://laravel.com/) with [Filament](https://filamentphp.com/) Template

## Install Dependencies

```
composer install
npm install
```

## Generate .env file

```
copy .env.example .env
```

## Generate App Key

```
php artisan key:generate
```

## Create a Filament User

```
php artisan make:filament-user
```

## Run the Server

```
php artisan serve
```
